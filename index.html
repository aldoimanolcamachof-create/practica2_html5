Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web. HTML5 es la versión más moderna del lenguaje HTML (HyperText Markup Language) y sirve principalmente para crear y estructurar páginas web. Es la base de casi todos los sitios que ves en internet.

Comenzar

¿QUE ES FOTOGRAFIA DIGITAL?

HTML5 es la versión más reciente, el cual se utiliza para crear y estructurar el contenido de las páginas web. Este lenguaje permite organizar elementos como textos, imágenes, enlaces y otros recursos dentro de un sitio web. Sirve principalmente para construir la estructura de una página web y facilitar la incorporación de contenido multimedia como videos y audios sin necesidad de programas externos. Además, incluye nuevas etiquetas que ayudan a organizar mejor la información, como encabezados, secciones, menús de navegación y pies de página, lo que hace que el código sea más claro y ordenado. Es importante porque es la base de todas las páginas web modernas. Gracias a HTML5, los sitios pueden adaptarse a diferentes dispositivos como celulares, tabletas y computadoras. También mejora el rendimiento de las páginas, permite una mejor experiencia para el usuario y facilita el trabajo de los desarrolladores al hacer el código más entendible y eficiente.

Estructura básica de HTML5

<!DOCTYPE html>
<html>
<head>
<title> Mi página</title>
</head>
<body>
</body>
</html>

!DOCTYPE html. .Es una declaración que se coloca al inicio del documento. .Su función es indicarle al navegador que el archivo está escrito en HTML5, para que lo interprete correctamente. .No es una etiqueta como tal, sino una instrucción. html. .Es la etiqueta raíz del documento. .Dentro de ella se encuentra todo el contenido de la página web. .Indica el inicio y el final del código HTML. head. Contiene información sobre la página que no se muestra directamente al usuario, como: .El título de la página title .Enlaces a archivos CSS .Metadatos (autor, codificación, descripción, etc.) Sirve para configurar y dar información al navegador sobre el sitio. body. Aquí va todo el contenido que sí se muestra en la página web, como: .Textos .Imágenes .Videos .Botones .Enlaces

SUS CARACTERISTICAS

h1 a h6

Son encabezados o títulos. h1 es el más importante (más grande) y h6 el menos importante.

p

Se usa para escribir párrafos de texto..

img

Se usa para insertar imágenes..

a

Sirve para crear enlaces a otras páginas o sitios web.

button

Sirve para que el usuario realice acciones, como: Enviar un formulario Ejecutar funciones con JavaScript Hacer clic para realizar alguna acción

ul / ol / li

Sirven para crear listas: ul lista desordenada (viñetas) ol lista ordenada (números) li cada elemento de la lista.

EJEMPLOS DE FOTOGRAFIA DIGITAL

header

Es el encabezado de la página o de una sección. Puede contener el logo, título o menú.

nav

Se usa para el menú de navegación. Aquí van los enlaces principales del sitio.

main

Contiene el contenido principal de la página. Solo debe haber uno por página.

section

Sirve para dividir el contenido en secciones. Cada sección trata un tema diferente.

article

Es contenido independiente, como: Noticias Publicaciones Blogs Se puede entender por sí solo.

footer

Es el pie de página. Contiene información como: Derechos de autor Contacto Redes sociales

GALERIA

¿Qué son los formularios en HTML? Los formularios son secciones de una página web que permiten a los usuarios ingresar y enviar información. Se usan, por ejemplo, para: Registrarse Iniciar sesión Enviar mensajes Llenar encuestas Etiquetas principales form → Contiene todo el formulario label → Texto que describe un campo input → Campo para escribir datos textarea → Para textos largos select → Lista de opciones button → Botón para enviar o hacer acciones

Conclusión

HTML5 es muy importante en el desarrollo web porque permite crear páginas más completas, organizadas y modernas. Gracias a sus etiquetas semánticas, es más fácil estructurar el contenido y hacerlo entendible tanto para las personas como para los navegadores. Además, facilita la integración de elementos como imágenes, audio y video sin necesidad de herramientas externas, lo que hace el desarrollo más sencillo y eficiente. En conclusión, HTML5 es una base fundamental para crear sitios web funcionales, accesibles y bien diseñados en la actualidad.
